## Project Description
**Explore Wisata** is a mini-program running inside the **DANA** app that lets users buy tickets to travel destinations around Bandung — browse, pick a destination, and pay without leaving the wallet.

## What I did
- Developed the mini-program UI (AXML structure, ACSS layout and styling)
- Built the business logic and pricing calculations in JavaScript
- Integrated the mini-program with the DANA API
