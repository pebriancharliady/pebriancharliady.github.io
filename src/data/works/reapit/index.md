## Project Description
**Reapit Sales** is a mobile app for real estate professionals: managing appointments, tasks, and listing-related data from the field. The app serves thousands of active users every day.

## What I did
- Led and built features used by thousands of daily active users
- Resolved user issues in close coordination with the QA and Product teams
- Maintained and produced production-ready builds for QA and release
