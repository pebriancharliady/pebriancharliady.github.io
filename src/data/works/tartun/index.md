## Project Description
**Tartun (Tarik Tunai)** is a mobile app that enables cash withdrawals and transfers through an **mPOS Cashlez device** — letting anyone run a cash-withdrawal point from their home or existing business.

## What I did
- Built the cash withdrawal and transfer features by integrating the Flip.id API with Java Servlets
- Developed the Android app UI and integrated it with the backend
- Built the web admin with JSP, JavaScript, and CSS on PostgreSQL 12
